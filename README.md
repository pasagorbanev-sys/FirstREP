#Универсал-дисконт
Мобильное приложение для программы лояльности с отображением персонального штрих-кода скидки.

##Функциональность
Авторизация — вход по коду доступа

Регистрация — создание профиля (ФИО, телефон, дата рождения)

Штрих-код — генерация и отображение скидочного штрих-кода

Профиль — просмотр личных данных и накопленного стажа

##Технологии
Язык: Java

Минимальный API: 21 (Android 5.0)

Архитектура: стандартная (Activity)

##Установка
Клонируйте репозиторий

Откройте проект в Android Studio

Соберите и запустите на устройстве или эмуляторе

##Использование
Тестовый код входа: 1234

Данные временно хранятся в памяти приложения

##Структура проекта
LoginActivity — экран входа

RegisterActivity — экран регистрации

BarcodeActivity — экран со штрих-кодом

ProfileActivity — экран профиля
